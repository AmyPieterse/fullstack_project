Routes description

HTTP Methods Path Description
GET / Takes you to homepage
GET /users Gives you the users from table users
GET /users/:id Retrieves a specific user based on id from table users
POST /register Adds a user to table users (Register)
PATCH/PUT /users/:id Updates user details based on id in table users
POST /login Checks if user details matches details in user table
DELETE /users/:id Deletes user based on id from table users
GET /products Retrieves all the products from table products
GET /products/:id Retrieves a specific product based on id from table products
POST /products Adds a product to the table
PUT/PATCH /products/:id Updates a product based on id to products table
DELETE /products/:id Deletes a product based on id from table
